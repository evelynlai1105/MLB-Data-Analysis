# MLB Data Analysis

MLB 打擊數據主成分分析

---

## Project Overview

本專案利用統計分析方法探討 MLB 球員打擊表現，透過主成分分析（PCA）找出影響打擊能力的重要因素。

---

## Business Problem

打擊成績受到多項指標影響。

如何利用統計方法降低變數維度，找出代表性指標，是本研究的主要目的。
